Данный проект направлен на изучение основ работы с базами данных в С#. Для этой цели использовались SQL запросы. 

Суть задачи заключалась в том, чтобы создать интерфейс для телефонного справочника, который по определенной информации будет выдавать подходящие данные, т.е. контакты людей. Пример его работы с данными о фамилии пользователя представлены на "Res1.png". В нижней части окна видим отсортированные данные, справа для наглядности видно автоматически сформированный запрос. Поиск по базе данных можно упрощать добавлением всей возможной информации (даже частично) "Res2.png", "Res3.png". Помимо выборки к запросу есть возможность добавить и сортировку данных "Res4.png".

В ходе реализации данной задачи были написаны различные методы для корректного формирования запроса. А также были написаны блоки методов для создания необходимых объектов интерфейса.
